# Campaign Automation System Architecture

## 🏗️ Overview

The campaign system follows a layered structure:

1. Creative Layer (Design assets)  
2. Template Configuration (HTML, WhatsApp templates)  
3. Automation Platforms (Customer.io, Interakt)  
4. Delivery APIs (WhatsApp, Email systems)  
5. Tracking & Analytics  

---

## 🔄 Flow

Creative → Template → Automation → API Delivery → Tracking
